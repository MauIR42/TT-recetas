reloj_tiempo_real
fecha de inicio sin fecha

resumen : usando el protocolo I2C y el disppsitivo DS3231 se hicieron las librerías para obtener la hora real sin importar la interrupción de corriente por un prolongado tiempo. Se utilizará este dispositivo para mandar JSON's al servidor con el tiempo en que se realizó la operación y así evitar inconsistencias.
Las librerías se lilstan a continuación:

	-DEFS: definicion de la estructura que contiene la informacion del reloj.

	- COMMUNICACION: en esta parte se maneja la creación de una tubería para escribir los datos a enviar entre el HILO y la aplicación principal.

	-HILO: uso de I2C al tener una lectura indefinida del reloj para siempre tener la hora a enviar al JSON, al recibir una interrupción por parte de la aplicación principal se escribirá en COMUNICACION la información de hora y fecha actual.

	-SIGNALS: maneja las señales de los hilos y exteriores como interrupción de la ejecución del programa.

	-SERVIDOR_RELOJ: programa principal en el que se inician las señales y el hilo, espera a que el usuario quiera saber la hora y la obtiene por medio de las señales, la tubería y el hilo.
